# Calculator

A simple calculator built with HTML, CSS, and JavaScript as part of The Odin Project Foundations curriculum.

## Features

* Addition, subtraction, multiplication, and division
* Supports decimal numbers
* Handles negative numbers
* Rounds decimal results to 4 decimal places
* Clear button to reset the calculator
* Input validation for invalid expressions

## Built With

* HTML5
* CSS3
* JavaScript (ES6)

## What I Learned

While building this project, I practiced:

* DOM manipulation and event handling
* Breaking functionality into small, reusable functions
* Managing application state
* Parsing user input and validating expressions
* Working with regular expressions
* Handling edge cases such as decimals, negative numbers, and division results with long decimal expansions

## Future Improvements
